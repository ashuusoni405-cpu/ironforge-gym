## 📌 Features

- ✅ User Registration / Login / Logout (localStorage based)
- ✅ 3 Workout Plans: Single Muscle, Double Muscle, Push Pull Leg
- ✅ Video Tutorials for each muscle group
- ✅ Membership Plans: Basic, Pro, Elite
- ✅ Dashboard with Workout Stats
- ✅ BMI Calculator
- ✅ Weight Tracker with Chart.js Graph
- ✅ Workout Log (Exercise, Sets, Reps, Weight)
- ✅ Water Tracker (8 glasses daily goal)
- ✅ Profile Page
- ✅ Fully Responsive (Mobile + Desktop)
